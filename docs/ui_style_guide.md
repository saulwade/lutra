# UI Style Guide

## Estilo general

Interfaz limpia, moderna y fácil de usar.

Inspiración:

- dashboards modernos
- apps de salud
- minimalismo clínico

## Colores base

verde suave  
beige claro  
blanco  
gris neutro  

## Componentes principales

### Sidebar

- Dashboard
- Pacientes
- Planes
- Recetas
- Configuración

### Dashboard

Resumen:

- pacientes activos
- planes recientes
- progreso

### Constructor de menú

Interfaz tipo tarjetas:

Desayuno  
Colación  
Comida  
Cena  

Cada comida muestra:

- alimentos
- porciones
- macros totales

### Buscador de alimentos

Barra de búsqueda rápida.

Resultados con:

- grupo alimenticio
- calorías
- equivalentes

### Vista de paciente

Panel con:

- datos clínicos
- historial de planes
- progreso
