# User Flows

## Flujo principal

### Crear un plan alimenticio

1. Nutriólogo inicia sesión
2. Abre un paciente existente o crea uno nuevo
3. Ingresa datos antropométricos
4. El sistema calcula requerimientos energéticos
5. Nutriólogo define macros
6. Sistema convierte macros a equivalentes
7. Nutriólogo construye menú
8. Sistema valida macros totales
9. Se genera vista previa
10. Se exporta PDF o link

## Flujo de paciente

1. Nutriólogo selecciona paciente
2. Abre historial
3. Ajusta dieta
4. Guarda nueva versión

## Flujo de alimentos

1. Nutriólogo busca alimento
2. Sistema revisa base interna
3. Si no existe:
   - consulta API externa
4. Nutriólogo puede guardarlo
